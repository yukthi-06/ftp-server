- Added new resource colors for button state handling:
  - `btnDisabledBackground` (`#E0E0E0` in Light, `#333333` in Dark)
  - `btnDisabledText` (`#8E8E93` in Light, `#777777` in Dark)
  - `btnTextEnabled` (`#FFFFFF` in both)
- Integrated `ColorStateList` in [MainActivity.java](file:///x:/Projects_X/0_Active/8_Android_APK/05_Early_Stage/FTP_Server_APK_GITHUB/app/src/main/java/com/vypeensoft/ftpserver/MainActivity.java).
- Updated [MainActivity.java:updateUI()](file:///x:/Projects_X/0_Active/8_Android_APK/05_Early_Stage/FTP_Server_APK_GITHUB/app/src/main/java/com/vypeensoft/ftpserver/MainActivity.java#L121) to programmatically style:
  - The **START SERVER** button:
    - Running state (Disabled): Disabled grey background, muted grey text, 0.6 alpha.
    - Stopped state (Enabled): Green background, white text, 1.0 alpha.
  - The **STOP SERVER** button:
    - Running state (Enabled): Red background, white text, 1.0 alpha.
    - Stopped state (Disabled): Disabled grey background, muted grey text, 0.6 alpha.
