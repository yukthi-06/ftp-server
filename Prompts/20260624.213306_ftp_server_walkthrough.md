# User Prompt
Create a complete Android application in **Java (NOT Kotlin)** named **FTP Server**.

## Objective

Build a lightweight FTP Server Android app that allows the user to start and stop an FTP server on their device and manage all configuration through a JSON settings file.

## Decisions & Verification

### Decisions Made
- **Storage Portability**: To handle Android 10+ storage restrictions, we request `MANAGE_EXTERNAL_STORAGE` (All Files Access) on API 30+ to access `/sdcard/Vypeensoft/FTP_Server/setings/settings.json` directly. We also built fallback mechanics in `SettingsManager` to query external storage directories dynamically.
- **In-Memory User Management**: Rather than generating static property files for FtpServer users, we implemented `org.apache.ftpserver.ftplet.UserManager` directly. This maps credentials checking directly to settings object.

### Verification
- Project layouts, resources, and Java classes have been written.
- The Gradle task `assembleDebug` can be executed within Android Studio or from terminal to compile the APK.
