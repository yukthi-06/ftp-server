# Plan - Add ACCESS_WIFI_STATE Permission

## Problem Description
The log shows a warning that the app does not have the `android.permission.ACCESS_WIFI_STATE` permission when `NetworkUtils` tries to retrieve the IP address from `WifiManager`. While it gracefully falls back to `NetworkInterface`, it is cleaner to declare the permission in the manifest to allow proper Wi-Fi status inspection.

### Android Manifest
