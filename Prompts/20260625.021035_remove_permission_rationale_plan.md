# Plan - Directly Request File Access Permission

## Problem Description
Currently, when requesting the "All Files Access" (`MANAGE_EXTERNAL_STORAGE`) permission on Android 11+ (API 30+), the app displays an explanatory `AlertDialog` before launching the system settings page. The user wants to remove this intermediate dialog and go directly to the permission request page.
