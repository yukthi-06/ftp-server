# User Prompt
Create a complete Android application in **Java (NOT Kotlin)** named **FTP Server**.

## Objective

Build a lightweight FTP Server Android app that allows the user to start and stop an FTP server on their device and manage all configuration through a JSON settings file.

# Tasks

- [x] Restructure Gradle configuration (`settings.gradle`, root `build.gradle`, app `build.gradle`)
- [x] Define AndroidManifest.xml with activities, service, boot receiver, and storage/network permissions
- [x] Create UI resources, layouts (`activity_main.xml`, `activity_settings.xml`), and string resources
- [x] Implement SettingsManager.java for JSON read/write logic at `/sdcard/Vypeensoft/FTP_Server/setings/settings.json`
- [x] Implement NetworkUtils.java for IP discovery, FTP URL formation, and port checks
- [x] Implement FTPServerManager.java to manage Apache FtpServer configuration and lifecycle
- [x] Implement FTPServerService.java to host the FTP server in a foreground service with a persistent status notification
- [x] Implement BootReceiver.java to automatically restart the FTP server if configured
- [x] Implement MainActivity.java to display FTP server status and start/stop controls
- [x] Implement SettingsActivity.java to edit settings.json configuration fields
- [x] Validate and run project assembly using Gradle (Attempted)
