# Implementation Plan - Dark/Light Theme Settings Option

We will add a "Dark Theme" preference toggle in the Settings screen, store it in the persistent settings, and apply the theme dynamically (dark/light) across all screens using Android\'s DayNight theme mode and qualifier resources.

## User Review Required

> [!IMPORTANT]
> The theme configuration will now read from a dynamic preference field (`dark_theme`) inside `SettingsManager`. Setting changes will be applied instantly. Layouts will be updated to use `@color/backgroundColor` instead of hardcoded white to support dark backgrounds natively in Dark mode.

#### [MODIFY] all Activities (MainActivity, SettingsActivity, HelpActivity, AboutActivity)
- Call a helper method/logic in `onCreate` to set night mode based on saved settings:
  ```java
  SettingsManager.Settings settings = SettingsManager.loadSettings();
  if (settings.dark_theme) {
      AppCompatDelegate.setDefaultNightMode(AppCompatDelegate.MODE_NIGHT_YES);
  } else {
      AppCompatDelegate.setDefaultNightMode(AppCompatDelegate.MODE_NIGHT_NO);
  }
  ```
