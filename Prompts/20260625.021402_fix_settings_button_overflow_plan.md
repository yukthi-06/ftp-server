# Plan - Fix Settings Buttons Text Overflow

## Problem Description
On the settings page, the "Reset Defaults" and "Save Settings" buttons have a fixed height (`56dp`) which can cause the text to overflow on certain screen sizes or under custom font settings.

- Change `android:layout_height` from `56dp` to `wrap_content` for both `@id/btnReset` and `@id/btnSave`.
- Add `android:minHeight="56dp"` to maintain the desired touch target size and aesthetic height.
