# Plan - App File Logging

## Problem Description
The user wants to collect application logs inside a file on `/sdcard/Vypeensoft/FTP_Server/logs/` directory. Each time the app starts, a new log file must be created with a timestamped format `ftpserver_yyyyMMdd.hhmmss.log`.
